# ココナラ出品案：ジャンプ力診断・トレーニングプラン作成サービス

jump_labの測定データを活用し、個別のジャンプ力診断とオーダーメイドトレーニングプランを提供する有料サービス。app単体では拾えない「パーソナル対応」の需要を収益化しつつ、jump_labへの導線にもする。

## 出品タイトル案
- 【jump_lab公式】あなたのジャンプ力を診断し、オーダーメイドの上達プランを作ります
- ジャンプ力測定データから、あなた専用のトレーニングメニューを設計します
- バスケ・バレー・陸上選手向け｜ジャンプ力診断＆パーソナルトレーニングプラン作成

## サービス概要（本文案）
```
「今のジャンプ力を、正しく知っていますか？」

jump_lab（ジャンプ力測定アプリ）の開発者が、あなたの測定データをもとに
現状を診断し、専用のトレーニングプランを作成します。

こんな方におすすめ：
✓ バスケ・バレー・陸上などジャンプ力が重要な競技をしている
✓ 自己流トレーニングで伸び悩んでいる
✓ 何を・どれくらいやればいいか分からない
✓ 体育教師でチーム指導に活かしたい

【提供内容】
1. jump_labでの測定データ（またはヒアリング）をもとにした現状分析
2. あなたの競技・目標に合わせたトレーニングプラン（4週間分）
3. フォームチェック用のポイント解説
4. 進捗確認・プラン調整（プランによる）
```

## プラン構成案

### ベーシックプラン（例: ¥3,000〜）
- 現状診断（測定データ分析）
- 汎用トレーニングメニュー（PDF）
- テキストでのアドバイス1往復

### スタンダードプラン（例: ¥6,000〜）
- ベーシックの内容
- 競技別・目標別にカスタマイズしたトレーニングプラン（4週間分）
- 週1回、進捗確認のメッセージ対応

### プレミアムプラン（例: ¥12,000〜）
- スタンダードの内容
- ビデオでのフォームチェック・フィードバック
- プラン期間中（1ヶ月）の質問し放題
- jump_labでの記録をもとにした月末振り返りレポート

## 進行の流れ
1. purchaseを見つけたら、jump_labのインストールと測定データ（またはヒアリングシート）を依頼
2. 3営業日以内に診断結果とプランを納品
3. スタンダード以上はメッセージで継続フォロー

## 差別化ポイント（説明文に入れる）
- アプリ開発者本人が直接対応する信頼感
- 感覚ではなくデータ（測定値の推移）に基づいた診断
- 「継続」を前提にした設計（jump_labのストリーク機能と連動した振り返り）

## 検討事項
- 価格帯・提供期間は要調整（他の類似出品をリサーチしてから確定）
- 「診断」を無料体験的に薄く出して、上位プランへのアップセルにする方式も検討可
